# Как собрать приложение для Android

## 1. Подготовка окружения

Вам потребуется установленный Flutter SDK и Android Studio (или Android SDK command-line tools).

1.  **Установите Flutter SDK:** [Инструкция](https://docs.flutter.dev/get-started/install)
2.  **Установите Android Studio:** [Скачать](https://developer.android.com/studio)
    - При установке убедитесь, что установлены компоненты: `Android SDK`, `Android SDK Command-line Tools`, `Android SDK Build-Tools`.
3.  **Настройте переменные окружения:** Добавьте путь к `flutter/bin` в переменную PATH вашей системы.

## 2. Сборка APK (установочный файл)

1.  Откройте терминал (PowerShell или CMD) в папке `ml_bot_app`.
2.  Установите зависимости Flutter:
    ```bash
    flutter pub get
    ```
3.  Запустите команду сборки release-версии APK:
    ```bash
    flutter build apk --release
    ```

После успешной сборки файл `.apk` будет находиться по пути:
`build/app/outputs/flutter-apk/app-release.apk`

## 3. Установка на телефон

### Способ 1: Через USB (ADB)
1.  Включите на телефоне **Режим разработчика** (Настройки -> О телефоне -> 7 раз нажмите на "Номер сборки").
2.  Включите **Отладку по USB** в меню "Для разработчиков".
3.  Подключите телефон к компьютеру кабелем.
4.  В терминале в папке проекта выполните:
    ```bash
    flutter install
    ```
    Это установит собранное приложение на подключенное устройство.

### Способ 2: Ручная установка
1.  Скопируйте файл `build/app/outputs/flutter-apk/app-release.apk` на телефон (через USB, Telegram, Google Drive и т.д.).
2.  Откройте файл на телефоне через файловый менеджер.
3.  Подтвердите установку из неизвестных источников (если потребуется).

## 4. Важные настройки

### Разрешения
При первом запуске (или при первом получении уведомления) Android может запросить разрешение на отправку уведомлений. Обязательно разрешите их, чтобы получать статусы от бота.

### Работа в фоне
Для корректной работы уведомлений в фоне на некоторых оболочках Android (MIUI, Samsung OneUI и др.) может потребоваться:
1.  Зайти в настройки приложения.
2.  Разрешить "Автозапуск".
3.  В настройках батареи выбрать "Нет ограничений" (чтобы система не убивала процесс).

### Сеть
Убедитесь, что телефон находится в одной сети с сервером бота (если используете локальный IP, например `192.168.1.X`), или что сервер доступен из интернета.
В настройках приложения укажите правильный URL API (например, `http://192.168.1.5:8765`).
