# Как собрать приложение для iOS (iPhone)

Поскольку у вас есть Mac с установленным Xcode, вы можете собрать и установить приложение на свой iPhone.

## 1. Подготовка файлов (на Windows)

Все необходимые файлы уже находятся в папке `ml_bot_app`.
Мы добавили необходимые настройки в `ios/Runner/Info.plist` (разрешение на HTTP запросы к вашему серверу), чтобы приложение работало корректно.

Вам нужно скопировать всю папку `ml_bot_app` на ваш Mac.
- Можно заархивировать в ZIP и переслать.
- Можно использовать Git (закоммитить изменения и склонировать на Mac).

## 2. Настройка на Mac

1.  **Установите Flutter SDK на Mac** (если еще нет).
    - [Инструкция по установке](https://docs.flutter.dev/get-started/install/macos)

2.  **Установите CocoaPods** (менеджер зависимостей для iOS):
    ```bash
    sudo gem install cocoapods
    ```

3.  **Откройте терминал на Mac** и перейдите в папку проекта `ml_bot_app`.

4.  **Установите зависимости Flutter:**
    ```bash
    flutter pub get
    ```

5.  **Установите зависимости iOS (Pods):**
    ```bash
    cd ios
    pod install
    cd ..
    ```

## 3. Сборка и запуск через Xcode

Это самый надежный способ, так как он позволяет легко настроить подпись приложения (Signing).

1.  Откройте файл `ios/Runner.xcworkspace` (белый значок) двойным кликом. Это откроет Xcode.
2.  Подключите ваш iPhone кабелем к Mac.
3.  В Xcode выберите ваш iPhone в качестве целевого устройства (вверху слева, где написано "Runner > Any iOS Device").
4.  **Настройка подписи (Signing):**
    - В левой панели выберите проект `Runner` (синий значок в самом верху).
    - В правой панели выберите вкладку **Signing & Capabilities**.
    - В разделе **Team**, выберите ваш **Personal Team** (Apple ID). Если его нет, добавьте свой аккаунт (Xcode -> Settings -> Accounts).
    - Убедитесь, что `Bundle Identifier` уникален (например, `com.yourname.mlbotapp`).

5.  **Запуск:**
    - Нажмите кнопку **Play** (треугольник) в верхнем левом углу Xcode.
    - Xcode скомпилирует приложение и установит его на телефон.

## 4. Доверьте приложению (на iPhone)

При первом запуске iOS может не дать открыть приложение от "Ненадежного разработчика".
1.  На iPhone зайдите в **Настройки -> Основные -> VPN и управление устройством** (или просто "Управление устройством").
2.  Нажмите на ваш Apple ID в списке "ПО разработчика".
3.  Нажмите **"Доверять <ваш email>"**.

## 5. Важные моменты

- **Сервер:** Убедитесь, что ваш iPhone и сервер (где запущен бот) находятся в одной сети, либо сервер доступен из интернета. В настройках приложения на телефоне укажите правильный IP адрес API (например, `http://192.168.1.X:8765` или внешний IP).
- **Фоновый режим:** Приложение будет получать уведомления, пока оно открыто или свернуто в памяти. Если iOS выгрузит его из памяти (свайп вверх), локальные уведомления перестанут приходить до следующего запуска.
